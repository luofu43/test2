# 线性回归模型训练项目

### 项目描述
完成y=wx+b线性回归模型的训练，分析权重w、偏置b与损失函数loss之间的关系。

### 实现功能
1. 使用pandas读取train.csv数据
2. 实现梯度下降算法训练线性回归模型
3. 绘制w-loss、b-loss关系图
4. 可视化训练过程和结果

### 文件说明
- `linear_regression.py`: 主程序文件
- `train.csv`: 训练数据
- `training_results.csv`: 训练过程记录
- `linear_regression_results.png`: 结果可视化图

### 运行结果
- 最终权重 w: 2.4710
- 最终偏置 b: 1.4851
- 最终损失: 3.5354
